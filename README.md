# South Bay Supercontest

## Stack

* Language: Python 3
* Application: Flask
* Server: uWSGI
* Proxy: nginx
* Database: PostgreSQL
* Frontend: Bootstrap 4
* API: GraphQL
* Migration: Alembic
* Composition: Docker
* Deployment: Ansible
